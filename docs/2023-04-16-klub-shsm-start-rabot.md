# Клуб ШСМ: старт работ

**Дата:** 2023-04-16

**URL:** http://systemsworld.club/t/topic/4221/1

![](upload://vnFNrCpjbpIyM3mdSop11Z4ExMf.jpeg)

Для того, чтобы проводить изменения на системных уровнях общества и человечества недостаточно усилий даже мощной компании. Требуется сообщество единомышленников, а чтобы появилось сообщество нужны активисты. Таких активистов хотим растить и объединить в клуб ШСМ. Ну а сам клуб будет создаваться сотрудниками ШСМ. Вот такая цепочка создания.

В клуб будут входить активные члены сообщества, которое сложилось вокруг ШСМ. Можно говорить о 7050 человек на 15.04.23 года, которые оставили нам свои контакты. В клубе будут состоять те из них, кто хочет объединиться в команды по созданию каких-то рабочих продуктов или просто проводить время в кругу единомышленников.

Недавно совместно с [@germiona\_o5](https://t.me/germiona_o5) провели [опрос](https://blog.system-school.ru/2023/04/02/research01/) по мероприятиям клуба, и приступаем к тестированию.

Например, в целях развития клуба решили ввести открытый формат защиты студентов курса «Системное саморазвитие». Защита будет проходить в [чате поддержки курса «Системное саморазвитие».](https://t.me/intro_systemsthinking) Там же планируем проводить тематические встречи. Например, обсуждать опыт постановки практик саморазвития, литературу подготовительного курса.

Клуб может быть площадкой для тестирования нашими студентами презентаций и публичных выступлений за рамками школы. Например, для амбассадоров или для работников, которым предстоит выступление на своем предприятии. Дополнительно будет обсуждать методики по практикам саморазвития:

- для новой редакции курса СС;
- курс для преподавателей школ и вузов, которые далее могут учить школьников и студентов;
- курс для школьников и студентов, который можно сделать общими усилиями или можно стимулировать наших выпускников сделать авторские курсы за рамками школы;
- для студентов ШСМ курс по экзокортексу.

В будущем у клуба будет отдельная страница на домене aisystant. Клуб будет межнациональным и общение будет на разных языках.
