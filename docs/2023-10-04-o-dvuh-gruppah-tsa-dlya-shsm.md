# О двух группах ЦА для ШСМ

**Категория:** Черновики|Церен Церенов (черновики)

**Дата:** 2023-10-04

**URL:** http://systemsworld.club/t/o-dvuh-gruppah-cza-dlya-shsm/5685/1

---

ЦА для подготовительных курсов — молодые, которые хотят быстро показать результат. Две группы ЦА -- студенты и молодые специалисты, которые активны (агентны) и хотят быстро получить старт, приносить пользу, встроиться в новый мир. А не только быстро заработать. Вторая группа -- это опытные специалисты, которые хотят сделать системную карьеру до уровня директора по развитию.
