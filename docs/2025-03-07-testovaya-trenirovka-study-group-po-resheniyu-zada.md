# Тестовая тренировка study group по решению задач

**Категория:** Новости

Завтра, 8 марта в 12:00 по Москве, пройдет тестовая тренировку study group по решению задач. Тренировка пройдет в чате [сообщества новой грамотности](https://t.me/intro\_systemsthinking/7534).
Про идею упражнений для изучения системного языка можно прочитать в посте ["Аналог Дуолинго для изучения системного языка"](http://systemsworld.club/t/analog-duolingo-dlya-izucheniya-sistemnogo-yazyka/22540). Правда, сейчас пока нет ИИ-агентов, а мы будем решать задачи из курса-тренажера, задействуя мышление письмом и "чувствуя" экран друга.
Тренировка будет проходить в формате одной помидорки — 25 минут. Запись будет доступна. Участие бесплатное, но необходимо иметь платную подписку ["Бесконечное развитие"](https://system-school.ru/openendedness). Требований по окончанию наших курсов нет.
Подробнее о формате study group можно прочитать по ссылке на пост в клубе "http://systemsworld.club/t/study-group-po-resheniyu-zadach/22669", а курс-тренажер можно найти по ссылке в https://aisystant.system-school.ru/lk/#/course/selfdev-trainer/2025-02-27T1338/60680.
