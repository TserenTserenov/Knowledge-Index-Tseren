Наталья, хорошая рефлексия, не хватает только описание гипотезы для тестировария.
Если разделяется гипотеза о важности режима дня, то значит его целесообразно нащупать так, чтобы слоты на саморазвитие и дальше другие Важные дела были максимально защищены. То есть найти наиболее вероятное для них время. Без этого вероятность саморазвития очень низка, поскольку оно либо систематично, либо никак.
